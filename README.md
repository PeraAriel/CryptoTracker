# CryptoTracker

CryptoTracker è un’applicazione web che permette di monitorare le principali criptovalute in tempo reale.  

Mostra per ogni crypto:  
- Nome e immagine  
- Prezzo attuale in USD  
- Variazione percentuale nelle ultime 24 ore (verde se positiva, rosso se negativa)  

L’app ha un tema scuro moderno e un layout a griglia responsive, con animazioni sulle card per una migliore esperienza utente.
